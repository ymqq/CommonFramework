# 模块说明
> 该模块负责Android相关基类编写，Activity、fragment、dialog等等基类编写。


> 该模块需要引用ModuleAndroidSupport模块。


> 模块中相关Version，全部在最外层build.gradle中定义，方便版本管理，确保使用一致的版本。
